# Blog Web App (Python3) 
A blog webapp demo developed in Python 3.5, including three parts: blogs, users and comments.

[demo link](http://52.37.25.30/)

### Requirements
1. Python 3.5.x
2. aiohttp
3. jinja2
4. aiomysql

### Front-end framework
[uikit](http://getuikit.com/)
